# CineStar Movie Booking API 🎬

A fully functional FastAPI backend system built as the final project for the **Innomatics Research Labs** FastAPI Internship. This project manages movies, seat holds, and ticket bookings using real-world API workflows.

## 🚀 Key Features
Implemented all 20 project tasks ranging from Beginner to Advanced:
* **Day 1 (GET APIs):** Home route, movie listings, and summary endpoints.
* **Day 2 (POST + Pydantic):** Data validation for bookings and new movie entries.
* **Day 3 (Helpers & Logic):** Specialized functions for cost calculation and filtering.
* **Day 4 (CRUD Operations):** Full Create, Read, Update, and Delete functionality for the movie database.
* **Day 5 (Multi-step Workflow):** A complete seat-hold and confirmation system.
* **Day 6 (Advanced APIs):** Implementation of keyword search, sorting, and pagination.

## 🛠️ Tech Stack
* **Language:** Python
* **Framework:** FastAPI
* **Validation:** Pydantic
* **Documentation:** Swagger UI (OpenAPI)

## ⚙️ How to Run
1. Clone the repository.
2. Install dependencies: `pip install -r requirements.txt`.
3. Start the server: `uvicorn main:app --reload`.
4. Open the interactive docs: `http://127.0.0.1:8000/docs`.
